# 智企连·AI-Ready

> 企业AI就绪度评估与提升平台

## 📖 项目简介

智企连·AI-Ready 是一个开源的企业AI就绪度评估与提升平台，帮助企业评估自身AI应用成熟度，并提供针对性的改进建议和实施路径。

## 🌟 核心功能

- **AI就绪度评估** - 全方位评估企业AI应用成熟度
- **差距分析** - 识别企业AI应用短板与改进空间
- **实施路线图** - 提供分阶段AI实施建议
- **最佳实践库** - 汇聚行业AI应用最佳实践
- **ROI预测** - AI投资回报率分析与预测

## 🚀 快速开始

### 环境要求

- Node.js >= 18
- Python >= 3.9
- Docker >= 20.10

### 安装

```bash
# 克隆仓库
git clone https://gitee.com/CozyNook/ai-ready.git
cd ai-ready

# 安装依赖
npm install
```

### 运行

```bash
npm run dev
```
