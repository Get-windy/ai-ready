|----|---------|--------|------|
| XSS-001 | HTML转义后事件处理器属性仍保留 (`onerror=`) | 中 | 🔴 需修复 |

### 修复建议

```python
# XSS清理需要移除事件处理器属性
def sanitize_html(input_str: str) -> str:
    import re
    # 先移除危险属性
    dangerous_attrs = r'on\w+\s*='
    cleaned = re.sub(dangerous_attrs, '', input_str, flags=re.IGNORECASE)
    # 再进行HTML转义
    for char, entity in {'<': '&lt;', '>': '&gt;', '"': '&quot;', "'": '&#x27;'}.items():
        cleaned = cleaned.replace(char, entity)
    return cleaned
```
