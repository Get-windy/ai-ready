## 二、熔断状态

```
CLOSED → OPEN → HALF_OPEN → CLOSED/OPEN
```

- CLOSED: 正常状态
- OPEN: 熔断，拒绝请求
- HALF_OPEN: 半开，允许部分请求测试

## 三、降级规则

熔断触发后返回默认响应或缓存数据。
