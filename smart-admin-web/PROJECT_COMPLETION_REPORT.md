## 技术栈

- **前端框架**: Vue 3.5.31
- **构建工具**: Vite 5.4.21
- **UI组件库**: Ant Design Vue 4.2.6
- **状态管理**: Pinia 2.3.1
- **路由管理**: Vue Router 4.6.4
- **HTTP客户端**: Axios 1.14.0
- **语言**: TypeScript 5.9.3
- **其他**:
  - dayjs（日期处理）
  - lodash-es（工具函数）
  - nprogress（进度条）
  - pinia-plugin-persistedstate（状态持久化）
  - vue-i18n（国际化）

## 代码特性

1. **TypeScript支持**: 所有代码使用TypeScript编写，提供类型安全
2. **组合式API**: 使用Vue 3 Composition API
3. **响应式设计**: 支持桌面端和移动端
4. **表单验证**: Ant Design Vue表单验证
5. **权限控制**: 基于路由元信息的权限验证
6. **错误处理**: 统一的错误处理机制
7. **代码规范**: ESLint代码检查
8. **组件化**: 高度组件化的代码结构
