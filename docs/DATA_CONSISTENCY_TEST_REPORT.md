## ⚠️ 数据一致性风险点

### 风险1: 读写分离延迟

**问题**: 主从复制可能导致读取延迟

**影响**: 中等

**建议**: 关键业务使用主库读取

### 风险2: 缓存穿透

**问题**: 大量无效请求穿透缓存

**影响**: 中等

**建议**: 实现布隆过滤器

### 风险3: 分布式事务

**问题**: 跨服务事务一致性保证复杂

**影响**: 高

**建议**: 使用Seata或Saga模式
